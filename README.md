# 🏖️ Bañito Pa Cuando?

## 🔧 Set up

1. Use correct Node.js version

```bash
nvm use
```

2. Install Dependencies

```bash
npm i
```

3. Husky pre-commit is installed after `npm i`

4. Run dev server

```bash
npm run dev
```

## 🧪 Tests

### Unit and integration testing

```bash
npm run test
```

### E2E Testing

```bash
npm run cypress:run
```

### Coverage

```bash
npm run test:coverage
```

## Format and linting

```bash
npm run lint
```

```bash
npm run format
```
